# Bem Viver Admin

Até aqui é um perfeito boilerplate pra iniciar projetos com CRA, TS, MUI e Styled Components

## Configuração do projeto

- Create-React-App
- Typescript
- Styled Components
- React Router DOM
- Material UI
- Webpack rewired by Craco and react-app-alias
  - To use styled components with Material UI
  - Absolute imports
